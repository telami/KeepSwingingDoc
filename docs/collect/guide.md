# 数据收集指南

## 手表端操作

1. 准备：
   - 打开「挥拍吧」app，点击左上角的设置按钮，开启**上传数据**开关（注意：平时打球时无需开启）

2. 数据采集：
   - 选择任意模式（如单打），按顺序采集四种基本挥拍动作：
     正手上手、正手下手、反手上手、反手下手
   - 每种动作重复30次，总计120次挥拍
   - 模拟真实比赛状态，每次挥拍之间要有移动和短暂停顿，保持自然流畅
   - 每完成一种动作后，点击"结束"，然后重新开始下一种

## 手机端操作

1. 在app设置中选择**开发者**选项，获取生成的.kw加密文件
2. 添加微信：lcy-cjdx，将文件发送给运营同事
